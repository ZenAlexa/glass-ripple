Why Glass Ripple

11-Pass Shader Pipeline

Wave simulation, normal mapping, Gaussian blur, compositing, dual halftone, chromatic aberration, CRT scanlines, vignette. All GPU-accelerated.

52 AI Brand Icons

Claude, GPT, Gemini, DeepSeek, Midjourney, Cursor, Vercel and 45 more. Every major AI brand, ready to drop in.

CRT Retro Aesthetics

Scanlines, phosphor glow, chromatic aberration, halftone dots. A cinematic, nostalgic feel that makes hero sections unforgettable.

3 Lines to Deploy

Import, create, done. Full TypeScript support, tree-shakable icons, works with any framework or vanilla JS.

Quick Start

import { GlassRipple } from 'glass-ripple'

const ripple = new GlassRipple({
  canvas: document.getElementById('canvas')
})
import { GlassRipple } from 'glass-ripple'
import { claude } from 'glass-ripple/icons'

const ripple = new GlassRipple({
  canvas: document.getElementById('canvas'),
  icon: claude.icon,
})
ripple.setTint('#D97757')
